Homework 10 - API Fun!

Please use the input boxes to see if there was a rocket launched on a particular date!

In the case of invalid input, the page will either not refresh, or indicate no launches.



Try: 27 August 2015, 16 July 1969, or 9 Auguts 2010 if you need some ideas for dates!

I used Launch Library's (link) API of historical rocket launches as my data source. It returns details of launches on a particular day, including, but not limited to: the model of rocket launched, the launch pad location, and the agencies or companies responsible for the launch. I found this API through APIlist.fun (link). I take the user's inputted day, month, and year and insert them into the following URL template for each API request: https://launchlibrary.net/1.3/launch/YYYY-MM-DD/YYYY-MM-DD. Additionally, I give the user another option, to check today's launches. This creates an API request that uses the same URL but does not necessitate user input. This API could be helpful for an avid fan of rockets to keep track of new launches, or computer scientists and mathematicians who need to know launch locations and rocket types to model flight paths of rockets.
